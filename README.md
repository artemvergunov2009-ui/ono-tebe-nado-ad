# Проект «Оно тебе надо»

Учебный проект по вёрстке сайта-аукциона вещей, в которые никто не верил.

## Ссылка на репозиторий
[https://github.com/artemvergunov2009-ui/ono-tebe-nado-ad.git](https://github.com/artemvergunov2009-ui/ono-tebe-nado-ad.git)

## Описание
В рамках проекта была реализована вёрстка по методу «прогрессивного джипега» с использованием CSS Grid и Flexbox.

## Технологии
* HTML5 (семантическая разметка)
* CSS3 (Grid Layout, Flexbox)
* Методология организации кода

## Как запустить
1. Склонируйте репозиторий или скачайте архив.
2. Откройте файл `index.html` в любом современном браузере.
